### Simple Multithreaded Hotmail Checker (SMHC) ###
- - -
####Quick brief ####
SMHC checks hotmail and MSN accounts list for availability to registration, free accounts goes to "good.txt" file. Lines should be in a "login@domain.com" format, one account per line.

![Screen](https://dl.dropbox.com/u/440947/cpp/SMHC-Screen.png)

- -  -
#### Features ####
* Multithreaded;
* Drag&Drop support;
* SSL support.

- -  -
#### Required tools and libraries ####

1. [Code Blocks IDE](http://www.codeblocks.org/downloads "Code Blocks")
2. [wxWidgets 2.9.4](http://wxwidgets.org/downloads/ "wxWidgets")
3. CURL with SSL support
 - Windows: use curl-7.27.0-devel-mingw32.zip from [here](http://curl.haxx.se/download.html "libcurl");
 - Linux: `aptitude install libcurl3 libcurl4-openssl-dev`
 

Install and configure IDE and libs, after that use CodeBlocks project files:

* Windows: **SMHC.cbp**;
* Linux: **SMHC-linux.cbp**.
